# AURA-IRS-Anomaly-Detection
AURA (Automated Unusual Return Analyzer) is an anomaly detection system designed to identify suspicious tax return patterns using data analysis and machine learning techniques.

1. **Data Preprocessing**

2. **Anomaly Detection**

   - Distance-based and density-based techniques
   - Threshold-based anomaly scoring

3. **Evaluation**
   - Statistical validation
   - Visualization of anomaly distributions
   - Comparison across multiple models
